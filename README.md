# Movie Library – Личная кинотека

**Автор:** Даниил Мартьянов *(укажите своё имя и фамилию)*

## Описание

Десктопное GUI-приложение на Python (Tkinter) для ведения коллекции фильмов.  
Позволяет добавлять фильмы, фильтровать по жанру и году выпуска, сохранять данные в JSON-файл.

## Функциональные возможности

- Добавление фильма с полями: название, жанр, год, рейтинг.
- Проверка корректности ввода (год – целое число, рейтинг – от 0 до 10).
- Отображение всех фильмов в таблице.
- Фильтрация по жанру (выпадающий список из уже добавленных жанров) и по году.
- Автоматическое сохранение в `movies.json`.
- Загрузка данных из JSON при запуске.
